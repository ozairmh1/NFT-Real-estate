# 📖 About The Project

Cryptic Estate is a futuristic blockchain-inspired real estate platform that allows users to invest in premium properties through fractional ownership.

Instead of purchasing an entire property, users can buy ownership tokens representing a percentage share of real estate assets. The project demonstrates how blockchain technology and tokenization can modernize traditional real estate investment systems.

The platform provides an immersive premium UI experience combined with wallet simulation, token verification, and dynamic investment tracking.
